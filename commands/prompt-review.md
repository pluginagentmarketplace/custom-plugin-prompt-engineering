# /prompt-review Command

Review and optimize prompts for better performance.

## Usage

```
/prompt-review "Your prompt text here"
/prompt-review path/to/prompt.txt
```

## What It Does

1. Analyzes prompt structure
2. Checks for common anti-patterns
3. Suggests improvements
4. Provides optimized version

## Output

- Clarity score (1-10)
- Identified issues
- Improvement suggestions
- Optimized prompt version
